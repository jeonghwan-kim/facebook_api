## Usage

1. facebook에서 앱ID key와 앱시크릿 코드를 발급 수령 후 `facebook-api-key.php` 에 입력
2. `facebook-redirect.php` 실행
3. 예제 파일 (`facebook-example-1.php`, `facebook-example-2.php`) 테스트
